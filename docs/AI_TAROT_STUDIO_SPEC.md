# AI Tarot Studio Product Spec

## Product vision

Arcarna Studio helps creators produce Tarot, Oracle, Angel, Lenormand, numerology, and philosophy-inspired decks without prompt-engineering expertise.

## User journeys

- Upload a reference image to extract style, lighting, palette, texture, and composition.
- Write an inspiration prompt and let the prompt builder expand it into production-ready visual language.
- Select AI Inspire Me to generate a concept from curated spiritual themes.
- Preserve Deck DNA across a full card set for consistent borders, palette, motifs, and character identity.

## Feature roadmap

1. Card creation workspace
2. Deck builder and card grouping
3. Gallery and iteration history
4. Marketplace-ready export tools
5. Premium provider switching and image enhancement

## Monetisation model

- Free tier for prompt previews and limited generations.
- Creator tier for Deck DNA, high-resolution exports, and saved galleries.
- Studio tier for premium FLUX Pro generation, card-back creation, and marketplace publishing tools.
