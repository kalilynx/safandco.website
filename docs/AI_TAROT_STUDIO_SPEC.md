# AI Tarot Studio Specification

## Product vision

Arcana Studio helps creators generate cohesive tarot, oracle, angel, Lenormand, numerology, and hybrid card decks through a guided creative workflow. The product hides prompt-engineering complexity behind curated controls, reference-image analysis, Deck DNA, and AI prompt optimization.

## Core user journey

1. Create a deck or single card.
2. Choose a creation mode: reference image, prompt only, reference plus prompt, or AI inspiration.
3. Select generator type and artistic direction.
4. Apply a visual collection and character/deck consistency preferences.
5. Review the optimized hidden prompt.
6. Generate card artwork through the selected image provider.
7. Refine artwork with publication tools.
8. Save to deck, gallery, or marketplace draft.

## Feature roadmap

- Reference image upload and visual analysis.
- Smart style recommendations from prompt content.
- Deck DNA persistence for full 78-card deck consistency.
- FLUX Schnell default generation with Stable Diffusion fallback and FLUX Pro premium tier.
- Print-ready export previews with safe zones, borders, and CMYK simulation.
- Marketplace metadata, attribution, and creator review workflow.

## Monetisation model

- Free tier: limited monthly generations and basic styles.
- Creator tier: Deck DNA, reference image analysis, higher resolution exports.
- Studio tier: premium providers, commercial exports, marketplace publishing, and batch deck generation.
