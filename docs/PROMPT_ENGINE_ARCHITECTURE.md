# Prompt Engine Architecture

## Prompt layers

1. User vision
2. Card system symbolism
3. Style rule injection
4. Reference image analysis
5. Deck DNA consistency
6. Provider-specific formatting
7. Negative prompt safety and quality rules

## Reference image processing

The analysis layer should extract lighting, camera angle, colour harmony, costume direction, facial proportions, texture, contrast, and visual hierarchy while avoiding direct copying.

## Deck consistency rules

Deck DNA stores recurring motifs, border language, palette, brushwork, typography-safe zones, aspect ratio, and visual rhythm for reuse across cards.
